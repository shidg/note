# 底行模式下
:1,4 m 6  #把1-4行移动到第6行之后[含1，4]
:1,4 co 6  #把1-4行复制到第6行之后[含1，4]
:1,4 d     #删除1-4行[含1，4]

:g/^/m 0 #倒序文件行
:g/^$/d #vi中删除空行
:g/^/+1 d #删除偶数行 也可以用normal命令 :%normal jdd
:g/^/d|m. #删除奇数行

:!ls #执行外部命令
:r !ls #将外部命令的执行结果写入到vim中



# 编辑模式下
*  读取光标处的字符串，并且移动光标到它再次出现的地方。
#  和上面的类似，但是是往反方向寻找。

c   行内删除
cc  删除整行并进入输入模式
cw  删除

zz 把当前置于屏幕中间，对应上下命令zt，zb

fx 移动光标到当前行的下一个 x 处。使用 ; 来重复上一个 f 命令。
tx 和上面的命令类似，但是是移动到 x 的左边一个位置。


# 编辑多个文件，vim -Oo  file1 file2 #垂直分屏或水平分屏
# 同已vim中多个文件间的复制粘贴可以使用v选择要复制的区域
# 使用寄存器
"f3Y   #复制3整行到寄存器f中

"fp    #将寄存器f中的内容粘贴到光标所在行之下，大写P则为粘贴到所在行之上

#系统剪切板
"+3yy  #复制三行内容到系统剪切板

# 快速删除全部内容

gg  # 定位到文件首行
dG  # 快速删除全部内容


J   #删除当前行末尾的换行符，也就是把下一行合并到本行末尾

C-R #重做，可以撤销u操作，也就是反撤销

:e!  #放弃所有修改并重新载入原始文件

``   #在两次所在行之间跳转(行的变换才视为跳转，但j、k除外)

C-0  #跳回到上次所在行

tab   #C-O的反操作

m[a-z] #可以定义26个标记，方便在文档的不同位置之间快速跳转

ma  #定义一个标记，名为a

`a  #跳转到标记a所在的位置(定义该标记时的精准位置)
'a  #跳转到标记a所在的位置(行首)

D  #d$ 删除当前光标到行尾
X  #dh 删除光标左边的一个字符
x  #dl 删除当前字符
C  #c$ 修改光标到行尾的内容
S  #cc 修改一整行
s  #cl 修改当前字符


. #normal模式下重复上次命令,这里的"命令"指的是导致文件内容发生变化的那些操作，比如cwhehe,d4w, df/等，而不包括查找(/  n)，定位等操作(f)


dw和daw的区别
dw删除光标到下一个word的开头
daw删除光标所在的整个word(含word之后的空白字符),不论光标在该词的何处
